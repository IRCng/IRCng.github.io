What is IRCng?

Let us make it clear what IRCng is. IRCng is a proposal that JSON be used for future implementations of an Internet relay chat system. The reference implementation, CaffeineIRCd, is written in PHP. It has minimal compatibility with the IRCv(2.7-3.0) protocols, in the form of use of a PROTOCTL command to enable the new (JSON-formatted) protocol.
